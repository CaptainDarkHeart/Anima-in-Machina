# LOOP LIBRARY TEMPLATE
## Cataloging Hypnotic Sections for Deep Space House Mixing

---

## PURPOSE

This template helps you identify, catalog, and rate loop-worthy sections in your deep space house collection. Use it to:
- Build a searchable database of extended mix opportunities
- Identify which loops work well together
- Plan strategic loop extensions during live/recorded mixes
- Remember which tracks have hidden gems for hypnotic extensions

---

## HOW TO USE THIS TEMPLATE

1. **Listen Through Track**: Play track fully at least once, noting moments that feel "loop-worthy"
2. **Identify Sections**: Mark start/end times of potential loop sections
3. **Test Loop**: Actually loop the section in Traktor/software to verify it works
4. **Rate Hypnotic Potential**: Score how well the section maintains interest when looped
5. **Document Texture**: Describe what makes this section loop-worthy
6. **Note Compatibility**: Identify which other tracks' loops could layer with this one

## LOOP RATING CRITERIA

Use these guidelines when rating hypnotic potential (1-10):

### 10/10 - Infinite Loop Territory
- Could genuinely loop for 5+ minutes without issue
- Has internal variation that prevents boredom
- Creates trance-like state
- Dancefloor loses track of time
- Example: Perfect dub techno groove, deep minimal techno loop

### 8-9/10 - Extended Loop Gold
- Solid for 2-4 minute extensions
- Maintains interest through groove or texture
- Subtle enough to avoid fatigue
- Works in appropriate context
- Example: Groovy percussion loop, atmospheric pad section

### 6-7/10 - Strategic Loop Use
- Good for 1-2 minute extensions
- Works for specific purposes (transitions, building tension)
- Needs exit strategy planning
- Context-dependent effectiveness
- Example: Breakdown sections, build-ups, specific textural moments

### 4-5/10 - Brief Loop Only
- 30-60 seconds maximum
- Gets repetitive quickly
- Best for quick extensions only
- Limited application
- Example: Vocal hooks, distinctive melodic phrases

### 1-3/10 - Not Loop-Worthy
- Boring or grating when repeated
- Too distinctive to loop
- Better left to play through once
- Don't bother setting cue point
- Example: Overly complex sections, vocal-heavy moments

## MONTHLY REVIEW PROCESS

Set aside time monthly to:

1. **Add New Discoveries**: 
   - Document 5-10 new loop sections from recent acquisitions
   - Test them in actual mix scenarios

2. **Update Ratings**: 
   - Revisit loops that didn't work as expected
   - Upgrade ratings for loops that proved valuable
   - Remove/downgrade loops that became boring

3. **Discover Compatibilities**:
   - Experiment with layering different loops
   - Document successful combinations
   - Note surprising failures for future reference

4. **Build Set Sections**:
   - Use loop library to plan extended journey sections
   - Identify gaps in your collection (e.g., need more minimal percussion loops)
   - Target new acquisitions to fill gaps
